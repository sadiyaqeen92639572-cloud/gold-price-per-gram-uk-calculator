# Gold Price Per Gram UK Calculator

Live UK gold price per gram calculator — 24k, 22k, 21k, 18k, 14k, 9ct and scrap gold, priced in GBP, updated three times daily from a live spot feed.

**Site:** [goldpricepergram.co.uk](https://goldpricepergram.co.uk)

## Structure

- `index.html` — pillar calculator (all purities).
- `24-carat-gold-price-per-gram-uk/`, `22-carat-gold-price-per-gram-uk/`, `21-carat-gold-price-per-gram-uk/`, `18-carat-gold-price-per-gram-uk/`, `14k-gold-price-per-gram-uk/`, `9ct-gold-price-per-gram-uk/` — one calculator per purity, each with a distinct use-case angle (investment, wedding jewellery, engagement rings, etc.) generated by `generate-pages.js`.
- `scrap-gold-price-per-gram-uk/` — scrap/selling estimate calculator.
- `methodology/` — data source, formula, and a live worked example.
- `site.config.json` — single source of truth for purity fractions, API config, dealer discount factor, and the page list.
- `update-data.js` — fetches the live spot price from [goldapi.io](https://www.goldapi.io/), computes per-purity prices, writes `gold-data.json`, and injects the result into every page via `<!-- START_PRICE_DATA -->` markers. Falls back to the last known price (with a visible banner) if the API is unavailable.
- `generate-pages.js` — regenerates the 8 satellite pages from `site.config.json` + inline content. Run this after editing page copy; run `update-data.js` after to re-inject current prices.

## Local testing

```bash
node update-data.js --mock          # inject a mock price without calling the real API
node update-data.js --force-fail    # test the fallback banner path
```

## Deploy

GitHub Pages via `.github/workflows/pages.yml` (deploys on push to `main`). Prices are refreshed by `.github/workflows/update-prices.yml` (cron, 3×/day), which commits and pushes `gold-data.json` + the regenerated pages — that push then triggers the Pages deploy.

Requires a `GOLDAPI_KEY` repository secret (free tier: 100 requests/month, well within the 3×/day = ~90/month schedule).
